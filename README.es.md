## Acerca de StayHard

Una aplicación móvil multiplataforma construida con Expo y React Native, que aprovecha Supabase para servicios backend y Google Gemini AI para funciones inteligentes.

## Funcionalidades

- Interfaz conversacional impulsada por IA con Gemini
- Sincronización de datos en tiempo real vía Supabase
- Navegación nativa con pestañas inferiores
- UI moderna con NativeWind y Heroicons
- Feedback háptico y animaciones
- Multiplataforma (iOS, Android, Web)

## Inicio rápido

### Requisitos previos

- Node.js 18+
- Expo Go (en dispositivo) o Android Studio / Xcode
- Cuenta de Supabase (para servicios backend)
- Google Gemini API key (para funciones IA)

### Instalación

```bash
git clone https://github.com/chrisssp/stayhard-mobile-app.git
cd stayhard-mobile-app
npm install
```

### Desarrollo

```bash
npx expo start
```

Escanea el código QR con Expo Go, o presiona `a` para emulador Android / `i` para simulador iOS.

| Capa | Tecnología |
